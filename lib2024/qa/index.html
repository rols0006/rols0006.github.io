Weekly QA

  1. What are the three components of Web?

    2. The three components are HTML, CSS, and JavaScript.

  2. Define version control and give one example.

    3. Version control is the ability to store previous version of code and recall them later. One example is v1 and v2.

  3. What tag is used to create the title of a web page? Give an example.

    4. The title tag is used. eg. <title> Weekly QA </title>

  4. What tag is used to create a paragraph in html? Give an example.

    5. The <p> tag is used to create paragraphs. eg. <p>This is a sample paragraph.</p>

  5. Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason.

    6. &<p> is the odd one out because it is the only non-semantic tag in this list.

  6. What are the tags required to create a navigation list in HTML?

    7. The tags <ul> and <ol> are used to create lists and <li> is used to create individual list elements.

  7. What element is used to create images? Name its attributes.

    8. The <img> tag is used to create images. <src> specifies the source of the image, <alt> specifies the alternate text for the image, <width> and <height> specify the size of the image.

  8. Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    9. <b> is the odd one out because it bolds font, the others are related to tables.
